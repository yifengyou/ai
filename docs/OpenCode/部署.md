# 部署

## 安装

OpenCode 的安装非常简便，支持多种操作系统和包管理器。

1.  **安装 (任选一种方式)**

    *   **通用 (npm)**
        ```bash
        npm install -g opencode-ai
        ```

    *   **一键安装脚本 (macOS/Linux)**
        ```bash
        curl -fsSL https://opencode.ai/install | bash
        ```
    *   更多安装方式（如 Docker、桌面应用）可参考[官方文档](https://opencode.ai/docs)。



## 配置

OpenCode 的配置文件是 opencode.json，通常位于以下路径：

```
    macOS / Linux: ~/.config/opencode/opencode.json
    Windows: C:\Users\你的用户名\.config\opencode\opencode.json
```


**示例一：配置阿里云百炼 (Bailian)**
```json
{
    "$schema": "https://opencode.ai/config.json",
    "provider": {
    "bailian-coding-plan": {
        "npm": "@ai-sdk/anthropic",
        "name": "Model Studio Coding Plan",
        "options": {
            "baseURL": "https://coding-intl.dashscope.aliyuncs.com/apps/anthropic/v1",
            "apiKey": "YOUR_API_KEY"
        },
        "models": {
        "qwen3.5-plus": {
            "name": "Qwen3.5 Plus",
            "modalities": { "input": ["text", "image"], "output": ["text"] },
            "limit": { "context": 1000000, "output": 65536 }
        }
        }
    }
    },
    "model": "bailian-coding-plan/qwen3.5-plus"
}
```

**示例二：配置京东云 JoyBuilder**
```json
{
    "$schema": "https://opencode.ai/config.json",
    "provider": {
    "joybuilder-plan": {
        "npm": "@ai-sdk/openai-compatible",
        "name": "jcloud Engine",
        "options": {
            "baseURL": "https://modelservice.jdcloud.com/coding/openai/v1",
            "apiKey": "YOUR_API_KEY"
        },
        "models": {
        "DeepSeek-V3.2": {
            "name": "DeepSeek-V3.2",
            "limit": { "context": 128000, "output": 64000 }
        }
        }
    }
    },
    "model": "joybuilder-plan/DeepSeek-V3.2"
}
```

* 修改并保存配置文件后，重启 OpenCode 即可生效。




## 运行

终端执行命令，进入终端会话

```
opencode
```
